# LEVEL 3 SOAL 10 (CRUD)
File ini terdiri dari file php, html, javascript dan database localhost dengan phpmyadmin dengan tujuan sebagai bahan pertimbangan untuk pihak Arkademy selaku penyelenggara test bootcamp web developer.

Syarat yang harus terpenuhi adalah:
1. Host: localhost
2. User: root
3. Password: '' (kosong)
4. Database: arkademy
5. Table: produk

Kolom tabel produk terdiri dari:
- nama_produk
- keterangan
- harga
- jumlah
